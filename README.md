# Random Quotes Project

## Description

A simple web-based application to present random quotes to user on a button click.

## Technologies Used

- HTML5
- CSS3
- JavaScript/JQuery
- Open Source Api by Rapidapi.com
- Git/GitHub - Github Pages for Hosting
