# Random Quotes Project

## Description

A web-based application to present random quotes to users on a button click. A funny way to start your day.

## Technologies Used

- HTML5
- CSS3
- JavaScript
- JQuery Modal
- Google Fonts
- Quotes Api by Rapidapi.com
- Git/GitHub - Github Pages for Hosting
